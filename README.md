#  Робот-помощник "Олег"

Мини-игра на **JavaScript** с использованием замыканий (*closures*) для создания сложного приватного состояния персонажа.

##  Описание
Главный герой — **Робот-помощник Олег**. Это трудолюбивый дроид, созданный для выполнения команд. Однако, чтобы он функционировал без сбоев, ему нужны:
* **Топливо** (сытость)
* **Вода** для охлаждения систем (вода)
* **Энергия** (отдых)

##  Правила и характеристики
Характеристики Олега ограничены диапазоном от `0` до `100`. Значения берутся из объекта конфигурации. 

**Каждое действие имеет последствия:**
* **Поесть:** +25% сытости, но тратит немного энергии.
* **Попить:** +20% воды.
* **Отдохнуть:** +30% энергии, но тратит сытость (-10%).

> **Внимание:** Ресурсы убывают со временем автоматически. Если они иссякнут, робот начнет терять здоровье. Смерть наступает при достижении `0`.

##  Реализованные усложнения (на оценку "5")
1.  **Настройка через конфиг и условные эффекты:** Все параметры (урон, лечение, затраты) вынесены в объект-конфиг. Если энергии `< 20`, робот "тупит" и ест менее эффективно.
2.  **Блокировка действий после смерти:** При статусе `isDead` кнопки интерфейса блокируются, а функции `eat()`, `drink()`, `rest()` возвращают ошибку.

## Запуск проекта
Просто откройте файл `index.html` в любом современном браузере. Выкачивать пакеты и запускать локальный сервер не требуется.
